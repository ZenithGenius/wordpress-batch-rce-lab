Desync playground

This is the batch desync (CVE-2026-63030) you can poke. Mark any sub-request's path as malformed and watch $matches lose a row, so every later request lines up with the wrong handler. Flip Apply the fix to add the one missing line and see the rows realign. Nothing here talks to a server; it runs the real array logic in your browser.
